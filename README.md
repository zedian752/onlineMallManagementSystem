

# 电商后台管理系统

## 描述

使用vue全家桶打造的线上商品后台管理系统

## 操作方式
```
npm install //安装所需包
npm run serve  //运行开发环境服务器
npm run build  //打包编译
```

## 分支

- master 主分支
- dist 打包编译后分支

## 文件描述

```
//src目录文件描述
```
